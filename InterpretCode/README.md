# Test Code in Line Interpreter

## php
```
$ php -a
php > echo "hello\n";
php > quit
```

## python
```
$ python
>>> print "hello"
>>> exit()
```

## perl
```
$ perl -e 'print "hello\n"'
```

## java
```
$ jshell
jshell> System.out.println("hello");
jshell> /exit
```

## javascript
```
$ node
> console.log(1);
> .exit
```
